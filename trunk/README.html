A free Solid State NMR simulation program.

What is pyPulsar?

PULSAR is a simulation program for solid-state nuclear magnetic resonance spectroscopy. pyPulsar is a user-friendly version of this program with a graphical interface and Python scripting capabilities)

Reports bugs

PyPulsar (version 0.1) is still in development stage. This is to say that there is absolutely no warranty of stability. It is delivered mainly for evaluation and debugging purpose.
